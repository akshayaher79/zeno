Blazing fast blogs for everyone

A lightweight, plugin-first Markdown → Blog framework built with JavaScript. Simple, hackable, and extendable.

Get Started → ⭐ Star on GitHub
Quick Start 
# Create a new blog
npx zeno-blog init mysite
cd mysite

# Build and serve
zeno build
zeno serve 3000
